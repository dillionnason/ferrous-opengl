# ferrous-opengl
A 3D game engine written in Rust using the OpenGL wrapper glium

## Building and running the project
```
git clone https://github.com/dillionnason/ferrous-opengl
cd ferrous-opengl
cargo build --release
./target/release/ferrous-opengl
```
